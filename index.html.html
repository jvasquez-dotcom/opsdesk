🔗 Conectar Google Sheets
1
Crea una Google Sheet
sheets.google.com → nueva hoja → nómbrala OpsDesk.
Crea 2 pestañas: tickets y usuarios.
2
Pega el Apps Script
Extensiones → Apps Script → borra todo → pega:
Luego: Guardar → Implementar → Nueva implementación → Aplicación web → Ejecutar como: Yo → Acceso: Cualquier persona → Implementar → copia la URL.
3
Pega la URL aquí
Al conectar, todos los datos locales se subirán a Sheets y la app sincronizará automáticamente cada 30 segundos.
✅ ¡Conectado! Datos subidos. La app sincroniza en tiempo real.
❌ No se pudo conectar. Verifica: (1) la URL es correcta, (2) el acceso es "Cualquier persona", (3) autorizaste el script.
